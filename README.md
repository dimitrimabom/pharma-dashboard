# pharma-dashboard

## Présentation

`pharma-dashboard` est un tableau de bord pour visualiser et analyser des données pharmaceutiques. Il fournit une interface claire pour suivre les indicateurs clés, les tendances et les performances.

## Installation

1. Cloner le dépôt :
   ```bash
   git clone https://github.com/dimitrimabom/pharma-dashboard.git
   cd pharma-dashboard
   ```
2. Installer les dépendances :
   ```bash
   npm install
   ```

## Utilisation

- Démarrer l'application en mode développement :
  ```bash
  npm start
  ```
- Construire pour la production :
  ```bash
  npm run build
  ```

## Structure du projet

- `src/` : code source de l'application
- `public/` : fichiers statiques
- `README.md` : documentation du projet
